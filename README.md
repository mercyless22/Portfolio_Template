### Run Locally

To run **Portfolio** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/mercyless22/Portfolio_Template.git
```

Windows:

```bash
git clone https://github.com/mercyless22/Portfolio_Template.git
```
